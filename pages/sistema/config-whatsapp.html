Configurações do WhatsApp (Conexões)

Motor de comunicação do sistema

O que é Config. WhatsApp?

É o motor de comunicação do sistema. Aqui geres as instâncias (números de telefone) que serão utilizadas para enviar e receber mensagens. O sistema permite múltiplas conexões simultâneas para escalar a tua operação, garantindo que possas atender mais clientes ao mesmo tempo com diferentes números.

Como Utilizar

1. Criar Nova Instância

Clica em "+ Nova Conexão". O sistema guiar-te-á num processo de 3 passos simples:

2. Passo 1: Configurar Instância

Preenche os dados básicos da conexão:

Nome da Instância

  • Dá um nome interno identificável (ex: "Suporte-Vendas", "Atendimento-Geral")
  • Regra: Usa apenas letras, números e hífens
  • Não pode conter espaços ou caracteres especiais
  • Exemplo válido: comercial-2024
  • Exemplo inválido: Comercial (Principal)

Número de Telefone

  • Insere o número que será conectado
  • Formato: Com código do país (ex: +351912345678)
  • Deve ser um número válido de WhatsApp
  • Não pode estar conectado em outro sistema simultaneamente
  • Recomendado: Número exclusivo para uso comercial

Provedor de API

  • Seleciona o provedor técnico que desejas utilizar
  • Opções disponíveis:
    • Evolution API: Solução não oficial, mais flexível
    • API Oficial Meta: Solução empresarial (configurar em módulo específico)
    • Outros provedores: Conforme disponibilidade

3. Opções de Automação

No momento da criação, podes ativar comportamentos automáticos para essa instância:

Ignorar Grupos

  • Quando ativado: Sistema não processa mensagens de grupos do WhatsApp
  • Útil para: Focar apenas em conversas individuais
  • Recomendado: Ativar se não usas grupos para atendimento

Sempre Online

  • Mantém o status de "visto por último" sempre ativo
  • Cliente vê: "online" constantemente
  • Benefício: Passa impressão de disponibilidade 24/7
  • Atenção: Pode gerar expectativa de resposta imediata

Marcar como Lido

  • Dá check azul (✓✓) automático nas mensagens recebidas
  • Cliente sabe: Mensagem foi visualizada
  • Útil para: Confirmar recebimento instantâneo
  • Cuidado: Cliente pode esperar resposta rápida após o check

Rejeitar Chamadas

  • Bloqueia chamadas de voz e vídeo automaticamente
  • Foco: Apenas comunicação por texto
  • Cliente recebe: Mensagem de chamada rejeitada
  • Recomendado: Para operações que trabalham exclusivamente com mensagens
  • Pode enviar mensagem automática: "Preferimos atendimento por texto"

4. Conectar (QR Code)

Após preencher os dados, o sistema gerará um QR Code nos passos seguintes:

Processo de Conexão

  1. Clique em "Gerar QR Code"
  2. Abre o WhatsApp no teu telemóvel
  3. Vai para WhatsApp → Menu (⋮) → Aparelhos Conectados
  4. Toca em "Conectar um aparelho"
  5. Aponta a câmara para o QR Code no ecrã
  6. Aguarda confirmação de conexão (geralmente 5-10 segundos)

Status de Conexão

  • 🟢 Conectado: Instância ativa e funcional
  • 🟡 Conectando: Aguardando leitura do QR Code
  • 🔴 Desconectado: Conexão perdida ou inativa
  • ⚠️ Erro: Problema na configuração

Reconexão

  • Se a conexão cair, clique em "Reconectar"
  • Novo QR Code será gerado
  • Repete o processo de leitura
  • Histórico de mensagens é mantido

5. Logs Webhook

Através do botão "👁️ Logs Webhook", podes monitorizar a comunicação técnica:

O que são Logs Webhook?

  • Registo de todas as comunicações entre WhatsApp e o sistema
  • Mostra eventos em tempo real (mensagens enviadas, recebidas, erros)
  • Útil para identificar falhas de conexão
  • Ferramenta de diagnóstico técnico

Informações Disponíveis

  • Timestamp: Data e hora exata do evento
  • Tipo de Evento: message.received, message.sent, connection.update
  • Status: Sucesso ou erro
  • Payload: Dados técnicos da requisição
  • Resposta: Retorno do servidor

Quando Usar

  • Mensagens não estão a ser entregues
  • Conexão cai frequentemente
  • Debugging de integrações
  • Suporte técnico solicita informações

6. Vincular a Agentes IA e Chatbots

Cada instância criada pode ser atribuída a um Agente de IA ou Chatbot específico:

Como Vincular

  1. Vai para Atendimento → Agentes IA ou Atendimento → Chatbots
  2. Ao criar ou editar um agente/chatbot
  3. Seleciona a instância desejada no campo "Número WhatsApp"
  4. Salva a configuração

Benefícios

  • Automação completa de atendimento
  • Cada número pode ter comportamento diferente
  • Segmentação por tipo de cliente ou serviço
  • Exemplo: Número 1 para vendas (Agente IA), Número 2 para suporte (Chatbot)

7. Gestão de Instâncias

Na tabela principal, visualiza e gere todas as conexões:

Colunas da Tabela

  • Nome: Identificador da instância
  • Número: Telefone conectado
  • Status: Estado da conexão (🟢🟡🔴)
  • Provedor: API utilizada
  • Criado em: Data de criação
  • Ações: Menu com opções

Ações Disponíveis (Menu "...")

  • Ver QR Code: Gera novo código para reconexão
  • Logs Webhook: Abre painel de monitorização
  • Editar: Altera configurações e automações
  • Desconectar: Encerra conexão (mantém configuração)
  • Excluir: Remove instância permanentemente

⚠️ Importante

  • Cada número só pode estar conectado em um sistema por vez
  • Não desconectes durante campanhas ativas (pode causar falhas)
  • Mantém o telemóvel com internet para estabilidade
  • Conexão pode cair se o telemóvel ficar offline por muito tempo
  • Recomendado: Usar número dedicado (não pessoal)

💡 Boas Práticas

  • Nomes Claros: Use nomes que identifiquem o propósito (ex: vendas-principal, suporte-24h)
  • Múltiplas Instâncias: Separe por departamento ou função
  • Monitorização: Verifica logs webhook semanalmente
  • Backup: Mantém sempre uma instância reserva configurada
  • Testes: Envia mensagem de teste após cada conexão
  • Documentação: Anota qual número está vinculado a qual agente/chatbot

Troubleshooting Comum

QR Code não funciona

  • Verifica se o número já está conectado noutro lugar
  • Tenta gerar novo QR Code
  • Confirma que o WhatsApp está atualizado

Conexão cai frequentemente

  • Verifica estabilidade da internet do telemóvel
  • Mantém o WhatsApp aberto em segundo plano
  • Evita desconectar outros aparelhos vinculados

Mensagens não enviam

  • Verifica status da conexão (deve estar 🟢)
  • Consulta logs webhook para erros específicos
  • Confirma que o número não foi bloqueado pelo WhatsApp
