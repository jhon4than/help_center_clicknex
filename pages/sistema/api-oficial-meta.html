API Oficial Meta

Integração profissional com WhatsApp Business API

O que é a API Oficial Meta?

É a integração profissional direta com o ecossistema da Meta (Facebook). Este módulo permite configurar o WhatsApp Business API (WABA), oferecendo maior volume de mensagens, métricas oficiais de qualidade, recursos avançados e integração com ferramentas externas como o RD Station. É a solução empresarial recomendada para operações de grande escala.

Como Utilizar

Pré-requisitos

Antes de começar, certifica-te de que tens:

1. Conta no Meta Business Suite

  • Acessa business.facebook.com
  • Cria ou acessa tua conta empresarial
  • Vincula página do Facebook (se aplicável)

2. Conta no Meta Developers

  • Acessa developers.facebook.com
  • Cria conta de desenvolvedor
  • Aceita termos e políticas

3. Empresa Verificada no Meta Business

  • Processo de verificação pode levar 1-3 dias úteis
  • Requer documentos oficiais da empresa
  • Necessário para usar API Oficial
  • Acessa: Meta Business Suite → Configurações → Verificação

4. Número de Telefone Válido

  • Número dedicado (não pode ser pessoal)
  • Não pode estar em uso noutro WhatsApp Business ou pessoal
  • Deve ser capaz de receber SMS ou chamada para verificação
  • Recomendado: Número novo exclusivo para API

1. Guia de Configuração

O sistema oferece um manual passo a passo integrado para te ajudar a navegar no painel de programadores da Meta:

Aceder ao Guia

  1. Clica no botão "📖 Guia de Configuração" no topo da página
  2. Segue as instruções detalhadas com capturas de ecrã
  3. O guia cobre:
    • Criação de App no Meta Developers
    • Configuração do WhatsApp Business
    • Obtenção de credenciais
    • Verificação do número

2. Nova Conexão (Credenciais)

Para conectar a API Oficial, deves obter e preencher no sistema as seguintes credenciais:

Phone Number ID

  • O que é: Identificador único do teu número no sistema Meta
  • Onde encontrar: Meta Developers → Teu App → WhatsApp → API Setup
  • Formato: Número longo (ex: 123456789012345)
  • Copia exatamente como aparece (sem espaços)

Business Account ID (WABA ID)

  • O que é: ID da tua conta WhatsApp Business
  • Onde encontrar: Meta Business Suite → WhatsApp Accounts
  • Formato: Número longo
  • Cada empresa tem um WABA ID único

Access Token (Token de Acesso)

  • O que é: Chave de autenticação para comunicação com a API
  • Onde obter: Meta Developers → Teu App → WhatsApp → API Setup → Token
  • Tipo: Usa "Token Permanente" (não temporário)
  • Formato: String longa alfanumérica
  • CRÍTICO: Nunca partilhes este token com terceiros
  • Guarda em local seguro (gestor de senhas)

App ID (Opcional)

  • O que é: Identificador da tua aplicação Meta
  • Onde encontrar: Meta Developers → Dashboard → Teu App
  • Necessário para recursos avançados

App Secret (Opcional)

  • O que é: Chave secreta da aplicação
  • Onde encontrar: Meta Developers → Configurações → Básico
  • Clica em "Mostrar" para revelar
  • Usado para validação de webhooks

3. Processo de Conexão

  1. Clica em "+ Nova Conexão API Oficial"
  2. Preenche todos os campos obrigatórios:
    • Phone Number ID
    • Business Account ID
    • Access Token
  3. Preenche campos opcionais se necessário (App ID, App Secret)
  4. Clica em "Testar Conexão" para validar credenciais
  5. Se teste passar (), clica em "Salvar"
  6. Sistema sincroniza automaticamente com Meta

4. Importar Conversas

Um recurso exclusivo que permite importar contactos e mensagens de outras plataformas:

Plataformas Suportadas

  • RD Station: Importa leads e histórico de conversas
  • HubSpot: Sincroniza contactos e interações
  • CSV Genérico: Importação manual via arquivo
  • Outras APIs: Conforme disponibilidade

Como Importar

  1. Clica no botão "📥 Importar Conversas"
  2. Seleciona a plataforma de origem
  3. Fornece credenciais de acesso (se necessário)
  4. Escolhe período de importação (últimos 30/60/90 dias)
  5. Seleciona tipos de dados:
    • Contactos/Leads
    • Histórico de mensagens
    • Tags/Etiquetas
    • Negócios/Oportunidades
  6. Clica em "Iniciar Importação"
  7. Aguarda processamento (pode levar minutos ou horas conforme volume)

Benefício

  • Não perdes histórico ao migrar para ClickNex
  • Continuidade no relacionamento com clientes
  • Dados consolidados num único lugar
  • Análises mais completas

5. Monitorização em Tempo Real

Na tabela principal, podes acompanhar métricas importantes:

Coluna Qualidade

Status de saúde do teu número perante a Meta:

  • Alta (Green): Excelente reputação
    • Poucas ou nenhuma reclamação
    • Limite máximo de envios
    • Sem restrições
  • Média (Yellow): Atenção necessária
    • Algumas reclamações de usuários
    • Limite reduzido temporariamente
    • Revê práticas de envio
  • Baixa (Red): Situação crítica
    • Muitas reclamações ou bloqueios
    • Limite severamente reduzido
    • Risco de suspensão
    • Ação imediata necessária

Coluna Limite (Throughput)

Quantas conversas podes iniciar por dia:

  • Tier 1: 1.000 conversas/dia (inicial)
  • Tier 2: 10.000 conversas/dia
  • Tier 3: 100.000 conversas/dia
  • Unlimited: Sem limite (casos especiais)
  • Tier aumenta automaticamente com bom uso
  • Pode diminuir se qualidade cair

Coluna Conversas

  • Contador de interações ativas nas últimas 24h
  • Mostra quantas conversas estão abertas
  • Útil para monitorar volume

Coluna Status

  • Ativo: Conexão funcionando normalmente
  • Sincronizando: Atualizando dados com Meta
  • Erro: Problema na conexão (verifica credenciais)
  • Suspenso: Número bloqueado pela Meta

Segurança

As credenciais são armazenadas de forma encriptada no sistema.

  • NUNCA partilhes o teu Access Token com terceiros
  • Não publiques credenciais em fóruns ou grupos
  • Se suspeitas de vazamento, gera novo token imediatamente
  • Usa tokens permanentes (não temporários de 24h)
  • Revoga tokens antigos após criar novos
  • Mantém App Secret em segredo absoluto

Boas Práticas para Manter Qualidade Alta

  • Opt-in: Envia mensagens apenas para quem autorizou
  • Conteúdo Relevante: Evita spam e mensagens genéricas
  • Responde Rápido: Clientes que iniciam conversa esperam resposta
  • Templates Aprovados: Usa apenas templates aprovados pela Meta
  • Horário Comercial: Evita envios fora do expediente
  • Monitorização: Verifica qualidade semanalmente
  • Feedback: Responde reclamações prontamente

Diferenças: API Oficial vs Não Oficial

API Oficial Meta (WABA)

  • Maior limite de mensagens
  • Suporte oficial da Meta
  • Menor risco de bloqueio
  • Recursos avançados (botões, listas, catálogo)
  • Métricas oficiais de qualidade
  • Processo de configuração mais complexo
  • Requer verificação empresarial
  • Custo por conversa (após tier gratuito)

API Não Oficial (Evolution, etc)

  • Configuração rápida (QR Code)
  • Sem verificação empresarial
  • Sem custos por mensagem
  • Risco de bloqueio maior
  • Sem suporte oficial
  • Limites mais baixos
  • Recursos limitados

Troubleshooting

Erro: "Invalid Access Token"

  • Verifica se copiaste o token completo
  • Confirma que é token permanente (não temporário)
  • Gera novo token no Meta Developers

Qualidade caiu para Média/Baixa

  • Revê últimas campanhas enviadas
  • Verifica se houve reclamações
  • Pausa envios em massa temporariamente
  • Foca em responder conversas iniciadas por clientes

Limite não aumenta

  • Mantém qualidade alta por 7 dias consecutivos
  • Aumenta volume gradualmente
  • Tier sobe automaticamente (não é manual)
