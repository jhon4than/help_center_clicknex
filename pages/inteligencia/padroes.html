O que são Padrões de Rastreamento?

É o configurador de "gatilhos" de rastreio. Aqui tu ensinas ao sistema que, quando um lead enviar uma mensagem específica (ex: vinda de um anúncio de Facebook), ele deve ser associado automaticamente a uma determinada campanha, conjunto de anúncios e anúncio. Elimina a necessidade de perguntar "por onde nos conheceu".

Como Utilizar

1. Cadastrar Novo Padrão

Clica em "+ Novo Padrão" e configura:

  1. Mensagem do Lead: Frase que o cliente envia
    • Insere a frase exata (ou similar)
    • Exemplo: "Olá, vim do anúncio e quero saber mais"
    • Exemplo: "Vi no Facebook sobre cozinhas"
    • Exemplo: "Quero orçamento - anúncio Instagram"
  2. O sistema detecta automaticamente variações similares

Dica de Configuração

Use a mensagem padrão que você configura no botão do anúncio. Exemplo: Se o botão diz "Quero saber mais", cadastre exatamente essa frase.

2. Associação de Dados de Tráfego

Preenche os campos com informações dos teus gestores de tráfego:

Campo: Campanha

  • Nome da campanha no Facebook Ads / Google Ads
  • Exemplo: "Campanha Black Friday 2024"
  • Exemplo: "Vendas - Cozinhas Planejadas"
  • Use o nome exato da plataforma de anúncios

Campo: Conjunto de Anúncios

  • Nome do conjunto (ad set) dentro da campanha
  • Exemplo: "Público Quente - Remarketing"
  • Exemplo: "Lookalike 1% - SP"
  • Ajuda a identificar qual segmentação funciona melhor

Campo: Anúncio

  • Nome específico do criativo/anúncio
  • Exemplo: "Vídeo Cozinha Moderna - V1"
  • Exemplo: "Carrossel Antes e Depois"
  • Permite testar diferentes criativos

3. Automação de Rastreio

Uma vez criado o padrão:

  • Qualquer lead que inicie conversa com essa frase será "etiquetado"
  • Sistema associa automaticamente aos dados de tráfego pago
  • Informações aparecem no perfil do lead
  • Dados alimentam o módulo de Rastreamento de Campanhas
  • Permite análise de ROI precisa

Importante

Configure padrões para cada anúncio diferente que você criar. Quanto mais específico, melhor a análise.

4. Testar Mensagem

Usa o botão "🧪 Testar Mensagem" para validar:

  1. Clique no botão de teste
  2. Digite uma frase que o cliente poderia enviar
  3. O sistema dir-te-á na hora se corresponde a algum padrão
  4. Mostra qual campanha/conjunto/anúncio seria associado
  5. Se não corresponder, você pode criar novo padrão

Exemplo de teste:

Mensagem testada: "oi vim do face quero orçamento"

 Correspondência encontrada!
Campanha: Vendas - Cozinhas Planejadas
Conjunto: Público Quente - Remarketing
Anúncio: Vídeo Cozinha Moderna - V1

Benefício Principal

Elimina a necessidade de perguntar ao cliente "por onde nos conheceu":

  • Dados 100% precisos: Sem depender da memória do cliente
  • Automático: Rastreio acontece em segundo plano
  • Escalável: Funciona para milhares de leads
  • BI Confiável: Relatórios baseados em dados reais
  • ROI Calculável: Sabe exatamente quanto cada anúncio gera

Exemplos Práticos

Padrão 1: Facebook Ads - Cozinhas

  • Mensagem: "Olá, vi no Facebook sobre cozinhas planejadas"
  • Campanha: Vendas - Cozinhas 2024
  • Conjunto: Interesse - Decoração
  • Anúncio: Carrossel Projetos Realizados

Padrão 2: Instagram - Promoção

  • Mensagem: "Vi a promoção no Instagram"
  • Campanha: Black Friday 2024
  • Conjunto: Lookalike Compradores
  • Anúncio: Stories - Desconto 30%

Padrão 3: Google Ads

  • Mensagem: "Encontrei no Google pesquisando cozinha planejada"
  • Campanha: Google Search - Cozinhas
  • Conjunto: Palavras-chave Exatas
  • Anúncio: Texto - Cozinha Sob Medida

Boas Práticas

  • Crie padrões específicos para cada anúncio importante
  • Use mensagens curtas e claras nos botões dos anúncios
  • Teste regularmente para garantir que funciona
  • Revise padrões quando criar novos anúncios
  • Mantenha nomenclatura consistente com gestor de tráfego
