O que são Projetos?

É o centro de gestão de entregas e backlogs. Aqui transformas negócios ganhos em planos de ação concretos, permitindo acompanhar prazos, tarefas concluídas e o progresso percentual de cada projeto. Converte oportunidades em execução organizada.

Como Utilizar

1. Criação Flexível

Ao clicar em "+ Novo Projeto", tens duas opções:

Opção A: Projeto em Branco

  • Para criar algo totalmente novo e personalizado
  • Começas do zero, sem tarefas pré-definidas
  • Ideal para projetos únicos ou experimentais
  • Adiciona tarefas conforme necessário

Opção B: A partir de Template

  • Replica uma estrutura de tarefas já usada anteriormente
  • Ideal para processos padronizados
  • Todas as tarefas do template são copiadas automaticamente
  • Economiza tempo em projetos recorrentes

Quando usar cada opção?

Projeto em Branco: Cliente com necessidades específicas
Template: "Onboarding de Cliente", "Implementação Padrão", "Projeto de Marketing"

2. Configuração do Projeto

Após escolher o tipo, preencha os dados essenciais:

  1. Nome do Projeto: Título claro e descritivo
    • Exemplo: "Implementação CRM - Empresa XYZ"
    • Seja específico para facilitar identificação
  2. Cliente/Empresa: Vincule ao cliente correspondente
    • Selecione da lista de organizações cadastradas
    • Ou crie um novo cliente diretamente
    • Vínculo permite acesso do cliente ao projeto
  3. Data de Início: Quando o projeto começa
    • Define o marco inicial
    • Usado para cálculos de prazo
  4. Data de Término: Prazo final de entrega
    • Estabelece deadline do projeto
    • Sistema alerta quando próximo do vencimento
  5. Clique em "Criar Projeto"

3. Acompanhamento Visual

A tabela principal exibe informações em tempo real:

Coluna Status

  • Ativo: Projeto em andamento (cor verde)
  • Pausado: Temporariamente suspenso (cor amarela)
  • Concluído: Projeto finalizado (cor azul)
  • Cancelado: Projeto não será concluído (cor vermelha)

Coluna Tarefas

  • Mostra formato: "X / Y"
  • X: Número de tarefas concluídas
  • Y: Total de tarefas do projeto
  • Exemplo: "8 / 12" = 8 tarefas concluídas de 12 totais

Barra de Progresso Automática

  • Calculada automaticamente: (Concluídas / Total) × 100%
  • Atualiza em tempo real conforme tarefas são marcadas
  • Representação visual colorida
  • Permite identificar rapidamente projetos atrasados

4. Ações de Gestão

No menu "..." de cada projeto, podes:

Editar

  • Alterar nome do projeto
  • Ajustar datas de início e término
  • Modificar cliente vinculado
  • Atualizar descrição e observações

Adicionar Tarefa

  • Insere nova tarefa diretamente naquele projeto
  • Atalho rápido sem sair da lista
  • Define título, responsável e prazo
  • Tarefa aparece automaticamente no quadro Kanban

Excluir

  • Remove o projeto do sistema permanentemente
  • Ação irreversível - todas as tarefas serão excluídas
  • Só é possível se não houver tarefas em andamento
  • Considere pausar ao invés de excluir

5. Filtros de Visualização

No topo da lista, use o seletor de filtros:

  • Todos os status: Exibe todos os projetos
  • Ativos: Apenas projetos em andamento
  • Pausados: Projetos temporariamente suspensos
  • Concluídos: Projetos finalizados
  • Atrasados: Projetos que passaram do prazo

Foco no que Importa

Use o filtro "Ativos" para focar apenas no que está em andamento e precisa de atenção.

Importante

  • Projetos sem tarefas mostram 0% de progresso
  • Ao pausar um projeto, as tarefas não aparecem no Kanban ativo
  • Projetos concluídos não podem ter novas tarefas adicionadas
  • A barra de progresso considera apenas tarefas não canceladas
