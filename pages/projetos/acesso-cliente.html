Acesso do Cliente

Permita que clientes acompanhem seus projetos

O que é Acesso do Cliente?

É o portal de transparência do teu sistema. Permite que geres um link seguro para que o teu cliente acompanhe, em tempo real, o andamento dos projetos dele sem precisar de ter uma conta de utilizador completa no sistema. Transparência que gera confiança e reduz solicitações de status.

Transparência

Cliente vê progresso em tempo real

Como Utilizar

1. Gerar Link de Acesso

Conceda acesso ao cliente de forma simples:

  1. Acesse Projetos → Acesso do Cliente
  2. Na lista de clientes, localize o cliente desejado
  3. Clique no botão "🔗 Gerar Link" ao lado do nome
  4. O sistema cria um link único e seguro instantaneamente
  5. Copie o link gerado
  6. Envie ao cliente por email ou WhatsApp

💡 Mensagem Sugerida

"Olá! Criamos um portal exclusivo para você acompanhar o andamento do seu projeto em tempo real. Acesse: [link]"

2. Validade do Link

Cada link gerado tem características de segurança:

  • Validade: 30 dias a partir da geração
  • Motivo: Questões de segurança e controle de acesso
  • Após expiração: Cliente não consegue mais acessar
  • Renovação: Gere um novo link quando necessário
  • Notificação: Sistema alerta quando link está próximo de expirar

⚠️ Segurança

Links expirados automaticamente garantem que apenas clientes ativos tenham acesso aos dados dos projetos.

3. Controlo de Status

Na tabela principal, visualize o estado de cada cliente:

Status: "Sem acesso"

  • Nenhum link foi gerado ainda
  • Cliente não tem acesso ao portal
  • Cor: Cinza
  • Ação: Clique em "Gerar Link"

Status: "Ativo"

  • Link foi gerado e está válido
  • Cliente pode acessar o portal
  • Cor: Verde
  • Mostra data de expiração

Status: "Expirado"

  • Link passou dos 30 dias de validade
  • Cliente não consegue mais acessar
  • Cor: Vermelho
  • Ação: Gere um novo link

4. Revogação de Acesso

Tens controlo total sobre os acessos:

  1. Na lista de clientes, localize o cliente com acesso ativo
  2. Clique no botão "Revogar" ou no menu "..."
  3. Confirme a revogação
  4. O link é cancelado instantaneamente
  5. Cliente perde acesso imediatamente

Quando revogar:

  • Projeto foi concluído e entregue
  • Cliente solicitou cancelamento
  • Suspeita de compartilhamento indevido do link
  • Mudança de responsável pelo projeto

O que o Cliente Vê

Ao clicar no link, o cliente tem acesso a uma visão simplificada e profissional:

Informações Visíveis

  • Lista de Projetos: Todos os projetos vinculados a ele
  • Progresso: Barra de conclusão percentual
  • Tarefas Concluídas: Lista do que já foi entregue
  • Tarefas em Andamento: O que está sendo feito agora
  • Prazos: Data de início e término previsto
  • Status Geral: Ativo, Pausado ou Concluído

Informações Ocultas (Privacidade)

  • Valores financeiros do projeto
  • Horas trabalhadas pela equipe
  • Tarefas marcadas como "internas"
  • Comentários privados da equipe
  • Custos e margens

Benefícios do Portal do Cliente

  • Transparência: Cliente vê progresso sem precisar perguntar
  • Reduz Solicitações: Menos emails e ligações pedindo status
  • Profissionalismo: Demonstra organização e controle
  • Confiança: Cliente se sente mais seguro com visibilidade
  • Satisfação: Experiência moderna e conveniente

⚠️ Importante

  • Cada cliente vê apenas seus próprios projetos
  • Links não podem ser transferidos entre clientes
  • Cliente não pode editar ou adicionar tarefas
  • Acesso é somente leitura (read-only)
  • Não é necessário criar login/senha para o cliente

💡 Boas Práticas

  • Gere o link logo após criar o projeto
  • Envie junto com mensagem de boas-vindas
  • Renove links antes de expirarem para clientes ativos
  • Use como diferencial competitivo nas vendas
  • Mantenha projetos atualizados para boa impressão
