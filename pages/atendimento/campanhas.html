Campanhas (Disparos API)

Envie mensagens em massa via API Oficial e Não Oficial do WhatsApp

O que são Campanhas (Disparos WhatsApp)?

É a ferramenta de comunicação em massa do sistema. Permite enviar mensagens para grandes listas de contactos (Leads ou Grupos) de forma automatizada e programada, com suporte a métricas de entrega e leitura. Alcance centenas ou milhares de clientes simultaneamente.

Como Utilizar

1. Criar Novo Disparo

Inicie uma nova campanha de mensagens:

  1. Clica em "+ Novo Disparo"
  2. Define um nome para a campanha:
    • Exemplo: "Promoção Black Friday"
    • Exemplo: "Lembrete de Pagamento - Novembro"
    • Exemplo: "Novidades do Mês"
    • Use nomes descritivos para facilitar identificação
  3. Clique em "Avançar"

2. Configuração de Envio

Escolhe o tipo de conexão que será usado:

Evolution API (Não Oficial)

  • Instância própria conectada via WhatsApp Web
  • Mais flexível para mensagens livres
  • Não requer templates aprovados
  • Ideal para comunicações informais
  • Pode enviar qualquer tipo de conteúdo

API Oficial Meta (WABA)

  • Conta Business verificada pela Meta
  • Exige uso de Templates pré-aprovados
  • Maior confiabilidade e entregabilidade
  • Menor risco de bloqueio
  • Ideal para comunicações corporativas

⚠️ Diferença Importante

API Oficial: Só pode usar templates aprovados pela Meta
API Não Oficial: Permite mensagens livres sem aprovação prévia

3. Agendamento e Delay

Configure quando e como as mensagens serão enviadas:

Tipo de Envio

  • Envio Imediato: Dispara assim que confirmar
    • Mensagens começam a ser enviadas instantaneamente
    • Útil para comunicações urgentes
  • Agendar: Programa uma data e hora futura
    • Escolha dia e horário específico
    • Sistema envia automaticamente no horário definido
    • Ideal para campanhas planejadas

Delay entre Mensagens (CRUCIAL)

É crucial definir o Delay entre mensagens em segundos:

  • Recomendado: 3-5 segundos entre cada envio
  • Por quê? Evita bloqueios do WhatsApp
  • Comportamento Humano: Simula digitação natural
  • Segurança: Reduz risco de banimento

💡 Configuração Segura

Para 1000 mensagens com delay de 5 segundos:
Tempo total: ~1h23min (muito mais seguro que envio instantâneo)

4. Selecionar Destinatários

Podes selecionar destinatários de três formas:

Opção A: Leads

  • Todos os Leads: Envia para toda a base de contactos
  • Seleção Manual: Escolhe leads específicos
    • Filtre por etiquetas
    • Filtre por origem
    • Filtre por responsável
  • Por Segmento: Use filtros avançados
    • Leads criados nos últimos 30 dias
    • Leads sem interação há X dias
    • Leads de uma região específica

Opção B: Grupos

  • Dispara para grupos de WhatsApp específicos
  • Selecione um ou múltiplos grupos
  • Útil para comunicados coletivos
  • Todos os membros do grupo recebem

Opção C: Manual

  • Insere números diretamente no campo
  • Um número por linha
  • Formato: 5511999999999 (DDI + DDD + número)
  • Útil para listas externas

5. Dashboard de Estatísticas

No separador "Uso / Limites" e no ícone de gráfico de cada campanha, podes monitorizar:

Métricas Principais

  • Total de Mensagens Enviadas: Quantidade total disparada
  • Entregues: Mensagens que chegaram ao destinatário
    • Indica sucesso na entrega
    • Percentual de entregabilidade
  • Lidas: Mensagens visualizadas pelo cliente
    • Disponível apenas na API Oficial
    • Indica engajamento real
  • Falhas: Mensagens que não foram entregues
    • Números inválidos
    • Bloqueios
    • Erros de conexão

Informações Temporais

  • Horário de Início: Quando o disparo começou
  • Horário de Conclusão: Quando terminou
  • Duração Total: Tempo decorrido
  • Taxa de Envio: Mensagens por minuto

6. Agendamento Recorrente (Dica de Segurança)

O sistema permite configurar campanhas periódicas automáticas:

  • Diário: Todos os dias no mesmo horário
  • Semanal: Escolha dias da semana específicos
  • Mensal: Dia específico de cada mês

Casos de uso:

  • Lembretes de pagamento mensais
  • Newsletter semanal
  • Bom dia diário para clientes VIP
  • Promoções de fim de semana

Como Utilizar

⚠️ Importante - Compatibilidade

As campanhas funcionam tanto para API Oficial (WABA) quanto para API Não Oficial. O processo é o mesmo, mas com algumas diferenças:

  • API Oficial: Requer templates pré-aprovados pela Meta
  • API Não Oficial: Permite mensagens livres sem aprovação prévia

Passo 1: Definir o Público-Alvo

Antes de criar a campanha, você precisa definir quem receberá as mensagens:

  1. Acesse Atendimento → Campanhas
  2. Clique em "Nova Campanha"
  3. Defina o público de 3 formas:
    • Lista de Contatos: Selecione uma lista pré-cadastrada
    • Importar CSV: Faça upload de arquivo com números
    • Manual: Digite os números manualmente

💡 Formato do CSV

O arquivo CSV deve conter uma coluna com os números no formato:

  • 5511999999999 (com DDI e DDD)
  • Sem espaços, traços ou parênteses
  • Um número por linha

Passo 2: Definir o Template da Mensagem

Configure a mensagem que será enviada:

Para API Oficial (WABA):

  1. Selecione um Template Aprovado pela Meta
  2. Preencha as variáveis do template (se houver)
  3. Visualize a prévia da mensagem

Para API Não Oficial:

  1. Digite a mensagem livremente no campo de texto
  2. Adicione variáveis personalizadas (opcional):
    • {{nome}} - Nome do contato
    • {{empresa}} - Nome da empresa
    • {{custom1}} - Campo personalizado
  3. Adicione mídias (imagem, vídeo, documento) se necessário

Passo 3: Configurar e Enviar

  1. Defina o tipo de envio:
    • Envio Imediato: Dispara assim que confirmar
    • Agendar: Escolha data e hora para envio
  2. Configure o intervalo entre mensagens (recomendado: 3-5 segundos)
  3. Revise todos os dados da campanha
  4. Clique em "Confirmar e Enviar"

Passo 3: Selecionar Destinatários

Você pode escolher os destinatários de 3 formas:

  • Lista de Contatos: Selecione uma lista pré-cadastrada
  • Importar CSV: Faça upload de arquivo com números
  • Manual: Digite os números manualmente

💡 Formato do CSV

O arquivo CSV deve conter uma coluna com os números no formato:

  • 5511999999999 (com DDI e DDD)
  • Sem espaços, traços ou parênteses
  • Um número por linha

Passo 4: Criar a Mensagem

Para API Oficial (WABA):

  1. Selecione um Template Aprovado pela Meta
  2. Preencha as variáveis do template (se houver)
  3. Visualize a prévia da mensagem

⚠️ Importante

Na API Oficial, você só pode usar templates previamente aprovados pela Meta. Não é possível enviar mensagens livres.

Para API Não Oficial:

  1. Digite a mensagem livremente
  2. Adicione variáveis personalizadas (opcional):
    • {{nome}} - Nome do contato
    • {{empresa}} - Nome da empresa
    • {{custom1}} - Campo personalizado
  3. Adicione mídias (imagem, vídeo, documento)

Passo 5: Configurar Envio

  • Envio Imediato: Dispara assim que confirmar
  • Agendar: Escolha data e hora para envio
  • Intervalo entre mensagens: Defina delay (recomendado: 3-5 segundos)
  • Horário de envio: Respeite horário comercial (8h-20h)

💡 Boas Práticas

  • Evite enviar fora do horário comercial
  • Use intervalo de pelo menos 3 segundos entre mensagens
  • Teste com poucos contatos antes de enviar para todos
  • Sempre ofereça opção de descadastro

Passo 6: Revisar e Enviar

  1. Revise todos os dados da campanha
  2. Verifique a quantidade de destinatários
  3. Confira a mensagem e variáveis
  4. Clique em "Confirmar e Enviar"

Acompanhamento da Campanha

Status da Campanha

  • Aguardando: Campanha agendada, aguardando horário
  • Em Andamento: Mensagens sendo enviadas
  • Concluída: Todos os envios finalizados
  • Pausada: Campanha pausada manualmente
  • Cancelada: Campanha cancelada antes de concluir

Métricas Disponíveis

  • Total de Envios: Quantidade de mensagens enviadas
  • Entregues: Mensagens que chegaram ao destinatário
  • Lidas: Mensagens visualizadas (apenas API Oficial)
  • Falhas: Mensagens que não foram entregues
  • Taxa de Entrega: Percentual de sucesso

Recursos Avançados

Personalização com Variáveis

Use variáveis para personalizar cada mensagem:

  • {{nome}} - Nome do contato
  • {{empresa}} - Nome da empresa
  • {{email}} - Email do contato
  • {{custom1}} a {{custom5}} - Campos personalizados

Exemplo:

Olá {{nome}}! 

Temos uma oferta especial para {{empresa}}.

Acesse: www.exemplo.com/{{custom1}}

Segmentação de Público

Crie listas segmentadas por:

  • Tags/Etiquetas
  • Funil de vendas
  • Última interação
  • Campos personalizados

⚠️ Políticas do WhatsApp

  • Não envie spam
  • Obtenha consentimento antes de enviar
  • Ofereça opção de descadastro
  • Respeite horários comerciais
  • Não envie conteúdo proibido (violência, drogas, etc)

Troubleshooting

Mensagens não estão sendo entregues

Possíveis causas:

  • Números inválidos ou bloqueados
  • Conta do WhatsApp banida
  • Template não aprovado (API Oficial)
  • Limite de envios atingido

Solução: Verifique o status da conta e os logs de erro

Campanha pausada automaticamente

Possíveis causas:

  • Taxa de falha muito alta (>30%)
  • Muitos bloqueios de usuários
  • Limite de API atingido

Solução: Revise a lista de contatos e aguarde 24h antes de tentar novamente
