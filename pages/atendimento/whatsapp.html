Central de atendimento multiagente via WhatsApp

O que é o Módulo WhatsApp?

É a central de comunicação em tempo real. Aqui, os operadores gerem todas as conversas vindas do WhatsApp, podendo alternar entre diferentes números e tipos de conexão (Oficial e Não Oficial). Interface unificada para atendimento multiagente profissional.

Como Utilizar

1. Seleção de Canal

No topo da barra lateral de mensagens, podes alternar entre as contas conectadas:

  • Clique no seletor de canal (ex: "Whats Oficial")
  • Visualize todas as instâncias de WhatsApp conectadas
  • Selecione o número que deseja atender
  • As conversas são filtradas automaticamente para aquele canal

Tipos de conexão disponíveis:

  • API Oficial (WABA): Conexão profissional via Meta
  • API Não Oficial: Conexão via WhatsApp Web/Evolution API

Múltiplos Números

Se sua empresa tem vários números de WhatsApp (vendas, suporte, financeiro), alterne entre eles sem sair da plataforma.

2. Status do Operador

Podes definir o teu estado de disponibilidade clicando no seletor de status:

🟢 Online

  • Disponível para receber novos atendimentos
  • Sistema distribui conversas automaticamente
  • Clientes veem que há operadores ativos
  • Notificações de novas mensagens ativadas

🟡 Em Pausa

  • Temporariamente indisponível
  • Não recebe novos atendimentos
  • Conversas atuais permanecem atribuídas
  • Útil para pausas, reuniões ou tarefas urgentes

⚪ Invisível

  • Visualiza conversas sem mostrar disponibilidade
  • Não aparece como operador ativo
  • Pode monitorar sem assumir atendimentos
  • Ideal para gestores ou supervisores

3. Meus Atendimentos

Filtra rapidamente as conversas atribuídas a ti:

  1. Clique no filtro "Meus Atendimentos"
  2. Visualize apenas conversas sob sua responsabilidade
  3. Foque na tua fila de trabalho sem distrações
  4. Priorize atendimentos pendentes

Outros filtros disponíveis:

  • Não Atendidas: Conversas aguardando primeiro contato
  • Em Atendimento: Todas as conversas ativas da equipe
  • Finalizadas: Atendimentos concluídos
  • Aguardando Cliente: Esperando resposta do cliente

4. Configurações de Conexão

Através do ícone de três pontos "⋮", tens acesso rápido a:

Config. QR Code (API Não Oficial)

  • Para conectar via WhatsApp Web
  • Escaneia o QR Code com teu celular
  • Conexão estabelecida em segundos
  • Ideal para testes ou pequenas operações

Config. API Oficial (WABA)

  • Gerir a conexão profissional via Meta
  • Configurar webhooks e tokens
  • Verificar status da conta Business
  • Gerenciar templates aprovados

Importante

Mantenha sempre uma conexão ativa. Se desconectar, você não receberá novas mensagens até reconectar.

5. Interface de Atendimento

As conversas aparecem na coluna central assim que os clientes enviam mensagens:

Coluna Esquerda: Lista de Conversas

  • Todas as conversas ativas
  • Ordenadas por última mensagem
  • Indicador de mensagens não lidas
  • Avatar e nome do cliente
  • Prévia da última mensagem

Coluna Central: Chat Ativo

  • Histórico completo da conversa
  • Campo de digitação na parte inferior
  • Botões para anexar mídia
  • Respostas rápidas (atalhos /)
  • Indicadores de entrega e leitura

Coluna Direita: Informações do Cliente

  • Dados cadastrais completos
  • Histórico de interações
  • Negócios vinculados
  • Tags e etiquetas
  • Notas internas da equipe

Boas Práticas de Atendimento

  • Responda em até 3 minutos (clientes esperam rapidez no WhatsApp)
  • Use o nome do cliente na conversa
  • Seja objetivo e cordial
  • Use emojis com moderação para humanizar
  • Sempre finalize com uma pergunta: "Posso ajudar em mais alguma coisa?"

Funcionalidades Principais

Respostas Rápidas

Crie atalhos para mensagens frequentes:

  1. Acesse Atendimento → Configurações → Respostas Rápidas
  2. Clique em "+ Nova Resposta"
  3. Defina:
    • Atalho: Ex: /horario
    • Mensagem: "Nosso horário é de segunda a sexta, 9h às 18h"
  4. Durante o atendimento, digite o atalho e pressione Enter

Transferência de Atendimento

Passe a conversa para outro atendente ou setor:

  1. Na conversa ativa, clique em "Transferir"
  2. Escolha o destino:
    • Atendente específico: Selecione pelo nome
    • Setor: Ex: Financeiro, Suporte, Vendas
  3. Adicione uma nota interna (opcional)
  4. Confirme a transferência

Notas Internas

Adicione observações que só a equipe vê:

  • Clique no ícone de nota () na conversa
  • Digite a observação
  • A nota aparece destacada em amarelo
  • Útil para registrar informações importantes ou contexto

Tags e Categorização

Organize conversas com tags:

  • Urgente: Requer atenção imediata
  • Reclamação: Cliente insatisfeito
  • Dúvida: Pergunta sobre produto/serviço
  • Venda: Oportunidade comercial

Crie tags personalizadas em Configurações → Tags

Filtros e Visualizações

Filtrar Conversas

Use filtros para organizar a fila:

  • Não Atendidas: Aguardando primeiro atendimento
  • Em Atendimento: Conversas ativas
  • Aguardando Cliente: Esperando resposta do cliente
  • Finalizadas: Atendimentos concluídos
  • Minhas Conversas: Apenas suas atribuições

Buscar Conversas

  1. Use a barra de busca no topo
  2. Busque por:
    • Nome do contato
    • Número de telefone
    • Conteúdo da mensagem
    • Tag aplicada

Regras Importantes

  • Não é possível dois atendentes responderem a mesma conversa simultaneamente
  • Conversas inativas por 24h são automaticamente finalizadas
  • Mensagens enviadas fora da janela de 24h (API Oficial) requerem template aprovado
  • Mantenha sempre o status atualizado para não confundir a equipe

Informações do Contato

Painel Lateral Direito

Veja informações completas do cliente:

  • Dados Cadastrais: Nome, telefone, email, empresa
  • Histórico: Todas as conversas anteriores
  • Negócios: Oportunidades de venda relacionadas
  • Projetos: Projetos em andamento
  • Tags: Categorias aplicadas
  • Campos Personalizados: Informações adicionais

Editar Informações

  1. No painel lateral, clique em "Editar"
  2. Atualize os dados necessários
  3. Clique em "Salvar"

Métricas e Relatórios

Indicadores de Performance

  • Tempo Médio de Primeira Resposta: Quanto tempo para responder novo contato
  • Tempo Médio de Atendimento: Duração média das conversas
  • Taxa de Resolução: % de atendimentos finalizados com sucesso
  • Satisfação do Cliente: Avaliação média recebida
  • Conversas por Atendente: Distribuição da carga de trabalho
