Etiquetas

Organize e categorize leads, negócios e contatos com tags

O que são Etiquetas?

Ferramenta de segmentação visual. As etiquetas servem para categorizar Leads e Negócios de forma personalizada, permitindo identificar rapidamente o perfil ou a necessidade de um contacto. São marcadores coloridos que facilitam a organização e priorização.

Visual

Cores vibrantes para identificação rápida

Segmentação

Filtre por perfil ou necessidade

Controlo

Veja quantos leads/negócios por tag

Como Utilizar

1. Criação Personalizada

Crie etiquetas que façam sentido para seu negócio:

  1. Clique no botão "+ Nova Etiqueta"
  2. Defina um nome descritivo:
    • "Hot Lead": Leads com alta probabilidade de compra
    • "Interessado em Cozinha": Segmentação por produto
    • "Urgente": Requer atenção imediata
    • "VIP": Clientes importantes
    • "Follow-up Semanal": Necessita acompanhamento
  3. Escolha uma cor vibrante:
    • Vermelho: Urgente, prioridade alta
    • Verde: Qualificado, pronto para comprar
    • Amarelo: Em análise, aguardando
    • Azul: Informativo, categorização
    • Roxo: VIP, especial
  4. Clique em "Salvar"

💡 Dica de Cores

Use um padrão de cores consistente em toda a equipe. Por exemplo: sempre use vermelho para urgência e verde para qualificado.

2. Adicionar Descrição

Documente o significado de cada etiqueta:

  • No campo "Descrição", explique quando usar a etiqueta
  • Adicione critérios de aplicação
  • Útil para alinhar a equipe
  • Aparece como tooltip ao passar o mouse

Exemplo:

Nome: Hot Lead
Cor: Vermelho
Descrição: Lead que demonstrou interesse concreto, 
respondeu perguntas de qualificação e tem orçamento 
definido. Prioridade máxima de follow-up.

3. Controlo de Uso

Na tabela principal, o sistema mostra estatísticas de cada etiqueta:

Coluna "Leads"

  • Quantidade de leads com esta etiqueta
  • Clique no número para ver a lista filtrada
  • Útil para identificar etiquetas mais usadas

Coluna "Negócios"

  • Quantidade de negócios marcados com a etiqueta
  • Permite análise de conversão por segmento
  • Identifique quais etiquetas geram mais vendas

💡 Análise de Performance

Compare quantos leads têm cada etiqueta vs quantos viraram negócios. Isso mostra quais segmentos convertem melhor.

4. Gestão de Etiquetas

Use o menu de ações "..." ao lado de cada etiqueta:

Editar

  • Altere o nome da etiqueta
  • Mude a cor para melhor organização
  • Atualize a descrição
  • Alterações aplicam-se a todos os registros existentes

Excluir

  • Remove a etiqueta do sistema
  • ⚠️ A etiqueta será removida de todos os leads e negócios
  • Ação irreversível - use com cuidado
  • Considere desativar ao invés de excluir

Aplicar Etiquetas

Como usar as etiquetas criadas:

Em Leads

  1. Abra o lead
  2. Clique em "Adicionar Etiqueta"
  3. Selecione uma ou múltiplas etiquetas
  4. As etiquetas aparecem coloridas no card do lead

Em Negócios

  1. No pipeline, clique no negócio
  2. Adicione etiquetas para priorização
  3. Útil para identificar negócios urgentes ou especiais

Filtros

  • Use filtros por etiqueta para segmentar visualizações
  • Crie campanhas direcionadas por etiqueta
  • Gere relatórios específicos por segmento

⚠️ Boas Práticas

  • Não crie etiquetas demais - mantenha entre 5-15 principais
  • Use nomes claros e objetivos
  • Documente o significado de cada uma
  • Revise periodicamente etiquetas não utilizadas
  • Padronize o uso com toda a equipe
