O que é o Pipeline de Negócios?

É o funil de vendas (CRM) onde geres as tuas oportunidades comerciais. Permite visualizar em que fase da venda cada cliente se encontra, desde o primeiro contacto até ao fecho do negócio. Uma ferramenta visual e intuitiva para acompanhar todo o processo de vendas.

Como Utilizar

1. Visualização em Pipeline (Kanban)

Os negócios aparecem organizados em colunas, cada uma representando uma etapa do funil:

  • Cada negócio é representado por um cartão visual
  • As colunas representam as etapas do processo de venda
  • Use drag-and-drop (arrastar e soltar) para mover negócios entre etapas
  • Basta clicar no cartão, segurar e arrastar para a coluna desejada

Dica Visual

Ao arrastar um negócio, a coluna de destino fica destacada. Solte o cartão na nova etapa para atualizar automaticamente.

2. Adicionar Novo Negócio

  1. Clique no botão "+ Negócio" no topo da tela
  2. Preencha os campos obrigatórios:
    • Título: Nome descritivo do negócio (ex: "Venda Cozinha - João Silva")
    • Produto/Serviço: Selecione do catálogo cadastrado
    • Valor: Valor estimado da venda em R$
  3. Adicione etiquetas coloridas para identificar prioridades:
    • Clique em "Adicionar Etiqueta"
    • Escolha cores como vermelho (urgente), verde (qualificado), etc
    • As etiquetas aparecem no cartão para identificação rápida
  4. Clique em "Salvar"

3. Personalizar Etapas do Funil

Adapte o pipeline ao seu processo de vendas:

  1. Clique no botão "+ Nova Etapa" no topo do pipeline
  2. Defina o nome da etapa:
    • Exemplos: "Qualificação", "Proposta Enviada", "Negociação", "Fechamento"
    • Use nomes que façam sentido para seu processo
  3. Escolha uma cor específica para a etapa:
    • Cores ajudam na identificação visual rápida
    • Sugestão: cores frias (azul) para início, cores quentes (verde) para fechamento
  4. Defina a ordem arrastando as etapas
  5. Salve as alterações

4. Usar Filtros de Pipeline

No topo da tela, você encontra o seletor de pipelines:

  • Pipeline Padrão: Funil principal de vendas
  • Outros Funis: Se você criou pipelines personalizados, aparecem aqui
  • Clique no seletor para alternar entre diferentes fluxos de venda
  • Útil para gerenciar produtos diferentes ou equipes separadas

Exemplo de Uso

Crie um pipeline "Vendas Corporativas" e outro "Vendas Varejo" para gerenciar processos diferentes com etapas específicas.

Dicas para Vender Mais

  • Mantenha o funil sempre atualizado (mova cards diariamente)
  • Defina valores realistas para ter previsão confiável
  • Registre todas as interações no histórico
  • Use lembretes para não esquecer follow-ups
  • Revise negócios parados há mais de 7 dias

Funcionalidades Principais

Visualização Kanban

Veja todos os negócios organizados por etapa:

  • Arraste e solte para mover entre etapas
  • Veja o valor total de cada coluna
  • Filtre por responsável, produto ou período
  • Clique em um card para ver detalhes

Visualização Lista

Veja negócios em formato de tabela:

  • Ordene por valor, data ou probabilidade
  • Edite valores diretamente na tabela
  • Selecione múltiplos negócios para ações em massa
  • Exporte para Excel ou CSV

Probabilidade de Fechamento

Cada etapa tem uma probabilidade padrão:

  • Qualificação: 10%
  • Proposta: 30%
  • Negociação: 60%
  • Fechamento: 90%

O sistema calcula automaticamente a Receita Prevista = Valor × Probabilidade

Gestão do Funil

Personalizar Etapas

  1. Acesse Sistema → Configurações → Funil de Vendas
  2. Clique em "Adicionar Etapa" ou edite existentes
  3. Defina:
    • Nome da etapa
    • Probabilidade de fechamento (%)
    • Cor de identificação
    • Ordem no funil
  4. Salve as alterações

Motivos de Perda

Quando um negócio é perdido, registre o motivo:

  • Preço muito alto
  • Escolheu concorrente
  • Não tem orçamento
  • Timing errado
  • Outros (especificar)

Isso gera insights valiosos sobre por que você está perdendo vendas.

Importante

  • Negócios parados na mesma etapa por mais de 30 dias são marcados como "Em Risco"
  • Você pode reabrir negócios perdidos se o cliente voltar
  • Negócios ganhos não podem ser editados (apenas visualizados)

Automações

Ações Automáticas

Configure automações para cada etapa:

  • Ao entrar na etapa: Enviar email, criar tarefa, notificar gestor
  • Após X dias parado: Enviar alerta, mover para outra etapa
  • Ao ganhar: Criar projeto, enviar contrato, notificar financeiro
  • Ao perder: Adicionar à lista de remarketing

Relatórios e Análises

Métricas Principais

  • Taxa de Conversão por Etapa: Quantos % avançam para próxima etapa
  • Tempo Médio por Etapa: Quanto tempo fica em cada fase
  • Ticket Médio: Valor médio dos negócios ganhos
  • Ciclo de Venda: Tempo médio entre criação e fechamento
  • Taxa de Ganho: % de negócios ganhos vs perdidos

Análise de Gargalos

Identifique onde seu funil está travando:

  • Etapas com muitos negócios acumulados
  • Etapas com baixa taxa de conversão
  • Etapas com tempo médio muito alto
